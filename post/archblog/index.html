archlinux windows10双系统安装过程

教程来源：知乎

1.联网

无线网络：

输入

iwctl

进入iwd模式，输入

device list

查看你的网卡名字，这里假设是wlan0，输入

station wlan0 scan

检查扫描网络，输入

station wlan0 get-networks

查看网络名字，假设名字叫BUPT-portal，输入

station wlan0 connect BUPT-portal

接着输入密码（如果有密码的话），输入

exit

退出iwd模式

连接成功之后，检查是否可以上网

ping www.baidu.com

重新设置mirrorlist（可选，建议）：

方法一：使用reflector来获取速度最快的6个镜像，并将地址保存至/etc/pacman.d/mirrorlist

reflector -c China -a 6 --sort rate --save /etc/pacman.d/mirrorlist

方法二：直接修改/etc/pacman.d/mirrolist

nano /etc/pacman.d/mirrolist

找到China复制几个到最上方即可

2.分区

1.检查硬盘

lsblk

2.建立分区

因为之前划好的空间在nvme0n1上，所以执行

cfdisk /dev/nvme0n1

新建swap分区，找到Free space选择new，输入分区大小，选择type未linux swap

（此时如果不想新建swap分区，可以参考5.正式配置新系统的建立swap文件）

新建根目录，找到Free space选择new，输入分区大小，选择Write，输入yes

写入完成 选择Quit 回车退出

开启交换分区

这里假定交换分区为nvme0n1p1

建立swap空间

mkswap /dev/nvme0n1p1

激活swap空间

swapon /dev/nvme0n1p1

3.分区格式化

将刚刚分好的区格式化为ext4格式，这里认准分区号是nvme0n1p2

mkfs.ext4 /dev/nvme0n1p2

4.挂载分区

先挂载/分区，同样，这里分区号也是nvme0n1p2

mount /dev/nvme0n1p2 /mnt

这里利用Windows的EFI分区，找到EFI分区号，这里假定是/dev/sdb3

建立boot文件夹

mkdir /mnt/boot

挂载EFI分区

mount /dev/sbd3 /mnt/boot

3.安装基本系统

执行

pacstrap /mnt base linux linux-firmware nano

等待安装完毕

（如果你不想用默认的内核，也可以使用linux-lts, linux-zen, linux-hardened，具体介绍请看Wiki）

4.生成fstab文件

genfstab -U /mnt >> /mnt/etc/fstab

检查生成的fstab文件

cat /mnt/etc/fstab

5.正式配置新系统

1.切换到装好的系统

arch-chroot /mnt

2.建立swapfile（建议，没有swap空间无法休眠）

在 ext4 上使用 swapfile 的用户请注意，升级到 5.7.x 内核后可能出现诸如「kernel: swapon: swapfile has holes」这样的报错而无法启用 swapfile 。使用 dd 命令创建 swapfile （而非 fallocate） 可能可以解决问题，也可以回退 5.6 系列内核等待上游修复。 Arch Linux 错误跟踪：https://bugs.archlinux.org/task/66921 内核错误跟踪：https://bugzilla.kernel.org/show_bug.cgi?id=207585

如果之前安装的内核是linux-lts：

fallocate -l 2GB /swapfile

注意：命令中是 小写字母l 而非 数字1 也非 字母i的大写

如果之前安装的内核不是linux-lts，这里创建swapfile需要使用dd命令：

dd if=/dev/zero of=/swapfile bs=2048 count=1048576 status=progress

这里分了2G作为swapfile

改权限

chmod 600 /swapfile

建立swap空间

mkswap /swapfile

激活swap空间

swapon /swapfile

修改/etc/fstab文件

nano /etc/fstab

到文件末尾输入

/swapfile none swap defaults 0 0

保存退出

3.设置时区

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

或

timedatectl set-timezone Asia/Shanghai

同步硬件时钟

hwclock --systohc

4.设置locale

nano /etc/locale.gen

Ctrl+W 输入 #en_US 回车 找到UTF-8那一行 删掉前面的#（取消注释）

Ctrl+W 输入 #zh_CN 回车 找到UTF-8那一行 删掉前面的#（取消注释）

保存退出

生成locale

locale-gen

创建并写入/etc/locale.conf文件

nano /etc/locale.conf

填入内容，注意这里只能填这个

LANG=en_US.UTF-8

5.创建并写入hostname

nano /etc/hostname

这里我写入的是 arch 作为hostname，你也可以输别的

保存退出

6.修改hosts

nano /etc/hosts

写入内容如图（中间的空白用tab而非空格），arch替换为你之前在hostname里写入的内容，其他都按照图里面的写（注意最后一行的ip是127.0.1.1）

127.0.0.1	localhost
::1			localhost
127.0.1.1	arch.localdomain arch

保存退出

建议上述编辑的内容都用cat输出检查一下

7.为root用户创建密码

passwd

然后输入并确认密码（linux终端的密码没有回显，输完直接回车就好）

8.创建启动器

安装基本的包，这里使用grub为启动器

pacman -S grub efibootmgr networkmanager network-manager-applet dialog wireless_tools wpa_supplicant os-prober mtools dosfstools ntfs-3g base-devel linux-headers reflector git sudo

如果你不知道这些包的作用，请务必确保输入的指令与上面的一致

检查完毕回车，需要选择直接回车就好，等待安装结束

输入

grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=Arch

确保输入指令完全正确回车没有报错

生成grub.cfg

grub-mkconfig -o /boot/grub/grub.cfg

9.退出新系统并取消挂载

exit

umount -R /mnt

重启

reboot

10.进入装好的Arch系统并激活网络

进去之后 先输入 root 回车 输入密码 回车

启动网络服务

systemctl enable --now NetworkManager

设置WiFi

nmtui

选择你要连接到的WiFi 输入密码 回车 然后退出

11.新建用户并授权

useradd -m -G wheel mir

wheel后面是你的用户名，这里输入的是mir

为用户创建密码

passwd mir

输入并确认密码

授权

EDITOR=nano visudo

Ctrl+W 输入 # %wheel 回车 找到这行 删除前面的 #（取消注释）

保存退出

12.安装显卡驱动

安装AMD集显驱动

pacman -S xf86-video-amdgpu

安装NVIDIA独显驱动

pacman -S nvidia nvidia-utils

13.安装Display Server

这里用的是开源世界最为流行的xorg

pacman -S xorg

出现选择直接回车即可

安装中文的字体：更纱黑体

pacman -S ttf-sarasa-gothic

等待安装完毕

重启

reboot

在grub界面选择archlinux回车即可
